# 🧬 Sistema de Criaturas Digitales

## 📌 Descripción

Este proyecto implementa un sistema basado en Programación Orientada a Objetos (POO) en Python utilizando una analogía de **criaturas digitales**.

El objetivo es demostrar el uso de:

* Clases abstractas (ABC)
* Herencia múltiple
* Encapsulamiento con propiedades (@property)
* Polimorfismo

Las criaturas digitales pueden comunicarse, evolucionar y gestionar su estado interno mediante atributos protegidos.

---

## 📁 Estructura del proyecto

```
nivelacion_python/
│
├── criatura_base.py
├── comunicacion.py
├── evolucion.py
├── criatura_digital.py
```

---


## 🧠 Funcionalidades principales

* Activación de criaturas (polimorfismo)
* Cambio de idioma y envío de mensajes
* Sistema de evolución con experiencia
* Control de estabilidad con validaciones
* Encapsulamiento usando getters y setters

---

## 🔒 Encapsulamiento

Se implementa mediante:

* Atributos protegidos (`_estabilidad`)
* Atributos privados (`__activa`)
* Uso de `@property` y `@setter` con validaciones
